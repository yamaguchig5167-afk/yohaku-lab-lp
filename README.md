# YOHAKU Lab. ランディングページ

**仕事を増やす前に、余白を増やす。**

YOHAKU Lab.（山口 剛）の公式ランディングページです。

## 技術構成

- React 18 + Vite
- GitHub Pages で自動デプロイ

## ローカルで確認する

```bash
npm install
npm run dev
```

## デプロイ

`main` ブランチにプッシュすると GitHub Actions が自動でビルド・デプロイします。

## vite.config.js の設定

リポジトリ名に合わせて `base` を変更してください。

```js
// リポジトリ名が yohaku-lab-lp の場合
base: '/yohaku-lab-lp/'

// カスタムドメインを使う場合
base: '/'
```
